## 🔐 How to Log into GitHub from Your PC

There are three common ways to authenticate with GitHub on your PC:

### Option 1: GitHub CLI (Recommended)
1. Download and install the [GitHub CLI](https://cli.github.com/)
2. Open a terminal and run:
   ```bash
   gh auth login
   ```
3. Follow the prompts — choose **GitHub.com**, then **HTTPS** or **SSH**, and authenticate via browser or token.

### Option 2: SSH Key
1. Generate an SSH key:
   ```bash
   ssh-keygen -t ed25519 -C "your_email@example.com"  # replace with your GitHub email
   ```
2. Copy the public key:
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
3. Go to **GitHub → Settings → SSH and GPG keys → New SSH key** and paste the key.
4. Test the connection:
   ```bash
   ssh -T git@github.com
   ```

### Option 3: HTTPS with a Personal Access Token
1. Go to **GitHub → Settings → Developer settings → Personal access tokens → Generate new token**.
2. Copy your token.
3. When Git asks for your password, paste the token instead.
4. To cache credentials securely, use the appropriate helper for your OS:
   - **Windows:** `git config --global credential.helper manager`
   - **macOS:** `git config --global credential.helper osxkeychain`
   - **Linux:** `git config --global credential.helper cache` (stores in memory temporarily)
